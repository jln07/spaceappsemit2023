For interacting with the data we can use Google Colab and Jupyter Notebooks; these are user-friendly tools that allow you to create and run code without needing expert programming skills.

Google Colab is free and open source; you can use it without any charges on your google cloud account.

Jupyter notebook is a web application that is free, open-source, and was developed as a spinoff of IPython. It enables users to create and share computation documents.

Follow the following steps:

  1. Navigate to the EMIT-Data-Resources a github
  2. Fork the repository.
  3. From the repository contents pick a jupyter notebook. For example: Exploring EMIT L2A Reflectancea
  4. Once you’re looking at the preview of the document download the raw file (This file will have an .ipynb extension)
  5. Go to and login with your google account
  6. Upload your . ipynb file
  7. Once it’s uploaded, double click on it and you will see the jupyter notebook running in google colab.
  8. You’re all set, now follow the instructions in the notebook and explore!
