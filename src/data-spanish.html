Poniendo los datos de la columna de metano de EMIT a tu alcance

Anteriormente, estos datos eran difíciles de consumir y comprendían aproximadamente 608 archivos. Usamos GenAI para escribir un script de Python que analizó estos 608 archivos JSON en un conjunto de datos coherente, que ponemos a disposición como un producto de datos aquí. Como siguiente paso, automatizaremos esto en una canalización de datos que procese continuamente datos EMIT y los agregueremos a este conjunto de datos.

Utiliza Google Colab para jugar con datos EMIT

Para interactuar con los datos podemos utilizar Google Colab y Jupyter Notebooks; Se trata de herramientas fáciles de usar que te permiten crear y ejecutar código sin necesidad de conocimientos de programación de expertos.

Google Colab es gratuito y de código abierto; puedes usarlo sin ningún cargo en tu cuenta de Google Cloud.

Jupyter notebook es una aplicación web gratuita, de código abierto y desarrollada como un derivado de IPython. Permite a los usuarios crear y compartir documentos con codigo.

Sigue los siguientes pasos:

  1. Navega a EMIT-Data-Resources GitHub
  2. Haz fork en el repositorio
  3. Selecciona una jupyter notebook del repositorio de contenidos, por ejemplo: Explorando EMIT L2A Reflectance
  4. Descarga el raw file (con extension .ipynb )
  5. Ve a Colab y entra con tu cuenta Google
  6. Sube el archivo .ipynb que descargaste
  7. Ahora da doble clic en el, ahora se puedes ver tu jupyter notebook en Google Colab
  8. Sigue las instrucciones en la jupyter notebook y explora!
