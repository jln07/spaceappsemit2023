Putting EMIT's methane plume data at your fingtertips

Previously this data was arduous to consume, comprising approximately 608 files. We used GenAI to write a python script that parsed these 608 JSON files into a cohesive dataset, which we make available as a data product here. As a next step, we'll automate this into a data pipeline that continuously processes EMIT data and adds to this dataset.

Use Google Colab to play with EMIT data

For interacting with the data we can use Google Colab and Jupyter Notebooks; these are user-friendly tools that allow you to create and run code without needing expert programming skills.

Google Colab is free and open source; you can use it without any charges on your google cloud account.

Jupyter notebook is a web application that is free, open-source, and was developed as a spinoff of IPython. It enables users to create and share computation documents.

Follow the following steps:

  1. Navigate to the EMIT-Data-Resources GitHub
  2. Fork the repository
  3. Pick a jupyter notebook from the repository contents, for example: Exploring EMIT L2A Reflectance
  4. Download the raw file (.ipynb extension)
  5. Go to Colab and log in with your Google account
  6. Upload your .ipynb file
  7. Once uploaded, double-click and you will see the jupyter notebook running in Google Colab
  8. Follow the instructions in the notebook and explore!
